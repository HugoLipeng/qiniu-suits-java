# 资源（音视频文件）查询元信息操作

# 简介
对空间中的音视频资源查询 avinfo 元信息。

### 配置文件选项
```
process=avinfo
domain=
use-https=
need-sign=true
```
`process=avinfo` 表示查询 avinfo  
`domain=` 表示用于查询 avinfo 的域名  
`use-https=` 是否使用 https 访问  
`need-sign=true` 源链接是否需要七牛私有签名  

### 命令行方式
```
-process=avinfo -domain= -use-https= -need-sign=true 
```
