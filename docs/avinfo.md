# 资源（音视频文件）查询元信息操作

# 简介
对空间中的音视频资源查询 avinfo 元信息。

### 配置文件选项
```
process=avinfo
domain=
https=
private=false
```
`process=avinfo` 表示查询 avinfo 操作  
`domain` 表示用于查询 avinfo 的域名  
`https` 是否使用 https 访问（默认否）  
`private` 资源域名是否为七牛私有空间域名（默认否）  

### 命令行方式
```
-process=avinfo -domain= -https= -private= 
```
